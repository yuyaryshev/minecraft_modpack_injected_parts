Custom_Event_Example Event Configuration
=======================================

This directory contains drops that only occur during the Custom_Event_Example event.

To activate this event, use the command:
/blockdrops event Custom_Event_Example true

To deactivate:
/blockdrops event Custom_Event_Example false

Configuration follows the same format as normal drops.
See config/EntityLootDrops/Blocks/Normal Drops/README.txt for full property documentation.
